# Project-PenPi2

Requirements:
  - Raspberry Pi
  - Kali ARM OS
  
  apt-get install
    wifite
    screen

Install:
  - Copy 'mailupdate.sh' to /root/Documents'
  - Copy all other files to /root/

Run:
  - Make files executable with chmod +x <file>
  - Run 'Startscreen.sh' > ./Startscreen.sh
